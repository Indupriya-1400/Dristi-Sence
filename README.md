🌊 Dristi Sense – Flood Prediction Tool
📌 Project Overview
Dristi Sense is a simple web-based flood analysis tool that predicts the possibility of floods using uploaded water or satellite images. The system analyzes wave patterns and water regions in the image and provides an estimated flood risk level.
This project demonstrates how image-based environmental monitoring can help in early flood detection and disaster preparedness.
🎯 Problem Statement
Floods cause severe damage to human life, infrastructure, and agriculture. Traditional monitoring methods require expensive equipment and manual observation.
There is a need for a simple and accessible system that can analyze water conditions from images and give early warnings about possible floods.
💡 Proposed Solution
Dristi Sense allows users to upload an image of water bodies such as rivers, waves, or satellite flood regions.
The system processes the image and provides a flood risk analysis, indicating whether there is:
No Flood Risk
Moderate Flood Probability
High Flood Risk
🛠 Technology Stack
HTML – Structure of the web application
CSS – User interface design
JavaScript – Image processing and flood analysis logic
⚙ How It Works
User uploads a water or satellite image.
The system previews the image on the interface.
The user clicks Run Flood Analysis.
The tool analyzes the image and generates flood statistics such as:
Estimated flood area
Affected population
Impacted districts
Prediction confidence
The result is displayed as a Flood Risk Status.
📊 Output Example
The tool displays information such as:
Flood Risk Status
Estimated Flood Area (km²)
Population at Risk
Affected Districts
Model Confidence Score
